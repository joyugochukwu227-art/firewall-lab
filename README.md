# Basic Firewall Configuration Lab

## Objective
To configure a basic firewall on a local Linux server using UFW and test access rules.

## Tools Used
- Ubuntu/Linux terminal
- UFW (Uncomplicated Firewall)

## Configuration Steps
- Default deny incoming traffic
- Allow SSH (22)
- Allow HTTP (80)
- Allow HTTPS (443)
- Enable firewall

## Files
- firewall-config.sh → firewall rules setup
- test-results.txt → verification of firewall behavior

## Outcome
Firewall successfully blocks unauthorized traffic while allowing essential services.
